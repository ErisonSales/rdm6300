# RDM6300 RFID reader Module, @125KHz
# Pinout:

----------------------------------------
|                        p1| 5 4 3 2 1 |
|                           -----------|
|                                      |
|_____                          _______|
| 1 2 |p2                    p3| 3 2 1 |
----------------------------------------

p1:
[#1]: TX
[#2]: RX
[#3]: NC
[#4]: GND
[#5]: +5v

p2:
[#1]: ANT1
[#2]: ANT2

p3:
[#1]: LED
[#2]: +5v
[#3]: GND
